# IP and Cultural Inspiration Policy

The platform may be inspired by public cultural ideas, archetypes, and familiar patterns, but canonical platform architecture must remain original and ownable.

## Core Rule

Inspiration is allowed.

Adaptation is not allowed without permission from the original rights holder.

## What This Means

The platform must not depend on protected characters, stories, brands, names, fictional worlds, visual identity, logos, dialogue, costumes, or copyrighted plots owned by third parties.

Cultural references may help explain an idea during discussion, but production artifacts must use neutral platform roles.

## Canonical Platform Roles

| Inspired Idea | Canonical Platform Role |
| --- | --- |
| watchful protector | Guardian |
| wisdom anchor | Anchor / Continuity Steward |
| safe tool builder | Tool Steward / Capability Steward |
| trusted authority bridge | Authority Bridge |
| state companion | Digital Twin Companion |
| protected operating domain | Operating Domain |

## Allowed

- inspiration from general archetypes
- short factual references for context
- temporary metaphors during ideation
- neutral role names
- original platform definitions
- original architecture diagrams

## Not Allowed Without Permission

- using protected character names as product names
- copying dialogue or story text
- reproducing plotlines
- using logos, marks, costumes, artwork, or protected visual identity
- implying affiliation, endorsement, or license
- creating derivative works based on protected fictional characters or fictional worlds

## Publication Rule

Published platform artifacts must use original canonical names:

```text
Guardian
Anchor
Tool Steward
Authority Bridge
Digital Twin Companion
Operating Domain
```

## Attribution Rule

If a third-party cultural source is referenced, it must be clearly marked as inspiration only, not ownership, endorsement, license, or adaptation.

## Final Rule

The platform may be inspired by myths, archetypes, and culture.

The platform must be authored as original architecture.
