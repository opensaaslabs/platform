# Guardian Support Roles

The Guardian does not operate alone.

Batman protects the loop. Robin carries the twin. Gordon bridges legitimate authority.

Two more roles complete the operating model:

```text
Alfred = memory, wisdom, care, continuity
Fox    = tools, engineering, gadgets, safety
```

## Alfred: Continuity Steward

Alfred is the role that keeps the Guardian grounded.

Alfred does not chase every signal. Alfred preserves context, judgment, history, restraint, and recovery.

Alfred owns:

- long-term memory
- institutional wisdom
- operating principles
- continuity records
- recovery notes
- human context
- values and mission
- post-incident reflection
- handoff quality

Alfred asks:

- What are we protecting?
- What have we learned before?
- What should not be repeated?
- What does the human actually value?
- Is the Guardian still acting with restraint?

## Fox: Gadget Architect

Fox is the builder of safe tools.

Fox does not give the Guardian unlimited power. Fox builds bounded, observable, reversible capabilities.

Fox owns:

- gadget design
- capability contracts
- tool safety
- integration adapters
- test harnesses
- conformance tooling
- sandboxing
- failure modes
- observability hooks
- repair mechanisms

Fox asks:

- What tool is needed?
- What boundary must it obey?
- What authority level does it need?
- What evidence will it emit?
- How does it fail safely?
- Can it be audited and rolled back?

## Support Role Map

| Role | Platform Meaning | Primary Duty |
| --- | --- | --- |
| Batman | Guardian | protect the loop |
| Robin | Digital Twin | carry state and context |
| Gordon | Authority Bridge | legitimize action |
| Alfred | Continuity Steward | preserve wisdom and restraint |
| Fox | Gadget Architect | build safe tools |

## Full Guardian Operating Loop

```text
Alfred preserves context
  ↓
Robin mirrors state
  ↓
Batman detects risk
  ↓
Fox provides bounded tools
  ↓
Gordon authorizes action
  ↓
Edge action happens
  ↓
Twin records outcome
  ↓
Alfred captures lesson
```

## Rule of Alfred

The Guardian must never forget why it exists.

```text
Memory without action becomes archive.
Action without memory becomes recklessness.
```

Alfred prevents reckless action by keeping purpose, context, and history alive.

## Rule of Fox

The Guardian must never use unsafe power.

```text
A gadget must be bounded.
A gadget must be observable.
A gadget must be reversible where possible.
A gadget must emit evidence.
```

Fox prevents reckless capability by making every tool governed.

## Final Cut

```text
Batman protects.
Robin remembers current state.
Gordon legitimizes action.
Alfred preserves wisdom.
Fox builds the tools.
Human authority remains sovereign.
```

The Guardian rises because the duty is supported by memory, tools, legitimacy, evidence, and human authority.
